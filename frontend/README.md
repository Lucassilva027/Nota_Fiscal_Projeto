# Frontend — Sistema de Nota Fiscal

Interface Angular do sistema de emissao de notas fiscais. Permite cadastrar produtos, criar notas, imprimir com baixa de estoque e simular falhas no servico de estoque.

## Servidor de desenvolvimento

```bash
npm start
```

A aplicacao fica disponivel em `http://localhost:4200/`.

## Build

```bash
npm run build
```

Os artefatos de producao sao gerados em `dist/`.

## Testes unitarios

```bash
npm test
```
